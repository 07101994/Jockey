Jockey   GitHub SourceForge

Jockey is a music player for Android

It's open source, built around simplicity, fast, and it follows Google's Material Design Guidelines

More Information

For more information including release notes, and to get in touch with the developer, please visit Jockey's GitHub Page.

Download Jockey

Jockey is currently in alpha, meaning that some features aren't implemented, rough around the edges, untested or horribly broken. If this sounds exciting, then you can download and install the compiled application from SourceForge, or compile it yourself from the source on GitHub.
