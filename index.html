Jockey

Jockey is a music player for Android. It's open source, built around simplicity, fast, and follows Google's Material Design Guidelines.

Screenshots

Get Jockey

Jockey is currently in alpha, meaning that some features aren't implemented, rough around the edges, untested or horribly broken. If this sounds exciting, then you can download and install the compiled application or build it yourself. All of Jockey's downloads are hosted on GitHub.
